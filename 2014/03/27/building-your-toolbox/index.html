Introducing DevCast: The Beginner's Toolbox

Posted on 27 Mar 2014 by Chapman Snowden and Myles Byrne

Tweet

Welcome to the first ever DevBootcamp DevCast! DevCast is a conversation about the beginner’s experience of learning how to learn, how to code, and how to build meaningful software.

In this episode, DBC’s Myles Byrne and Chapman Snowden talk about building your “toolbox” as a beginner web developer. You’re toolbox can mean a number of different things - mental strategies, resources, code snippets. But like any tool, the magic isn’t necessarily the tool, but in how you use it.

During the episode we reference this awesome talk, Programming with Hand Tools by Tim Ewald.

Let us know what you think or share topics you’d like to see covered in future episodes by tweeting to us @DevBootcamp!
